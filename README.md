# Airbnb-Dashboard

Global Airbnb Performance Dashboard

An interactive Power BI dashboard project focused on analyzing Airbnb market trends, customer ratings, seasonality, and host trust insights across multiple cities worldwide.

Features
1. KPI Overview
2. Listings Growth Analysis
3. City-wise Market Share
4. Ratings Heatmap
5. Seasonal Trend Analysis
6. Trust & Verification Insights

Tools Used
1. Power BI
2. DAX
3. Data Visualization
4. Business Intelligence

Key Insights
1. Airbnb listings peaked in 2015
2. Paris recorded the highest listings and reviews
3. Mexico City showed strong customer ratings
4. Seasonal trends significantly affected review activity

Project Goal

To transform raw Airbnb data into meaningful visual insights and improve business storytelling through interactive dashboards
